- Работаем в Codespaces с использованием плагина [live-server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) или любого другого плагина для хостинга статики
- Файлы создаются в корне репозитория
- Можно использовать bootstrap (файлы прилагаются)
- Выполненное задание необходимо закоммитить, скачать и положить по пути `Студенческий Диск/ИСР31/Экзамен` (Качаете код из репозитория `Code => Download Zip`, разархивируете и переносите на диск)
